Lego Tetris
===========

A browser Tetris game rendered with Lego-brick styled blocks (studs, bevels, highlights).

Open `index.html` in any modern browser to play.

## Controls

**Keyboard**
- `←` / `→` — move
- `↓` — soft drop
- `↑` / `X` — rotate clockwise
- `Z` — rotate counter-clockwise
- `Space` — hard drop
- `P` — pause

**Touch (iPhone / Android)**
- On-screen D-pad with dedicated Move / Rotate / Soft-drop / DROP buttons
- Hold left/right/down to auto-repeat
- On the board itself: tap to rotate, swipe horizontally to move, swipe down to soft-drop, flick down quickly to hard-drop

## Features

- Standard 10×20 playfield with the 7 classic tetrominoes
- 7-bag randomizer for fair piece distribution
- Ghost piece, hard/soft drop, wall kicks
- Next-piece preview
- Progressive speed (level increases every 10 lines)
- Score: 100/300/500/800 per 1/2/3/4 lines, multiplied by level
